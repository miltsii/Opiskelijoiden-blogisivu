# Opiskelijoiden-blogisivu
Sivulla pystyt kirjoittamaan omaa blogia, seuraamaan muiden opiskelijoiden blogia ja hakea tiettyjä blogeja hausta.
Pystyt kommentoida ja tykätä muista blogeista. Pystyt myös muokata oman profiilisi kuvaa ja nimeä. (Ehkä myös pystyt vaihtamaan koko sivun teemaa (pimeäksi tai valoisaksi) ja muuta hassua mitä en oo viel keksiny)
